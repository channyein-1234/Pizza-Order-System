# Pizza Order System
A full-stack food ordering system that allows customers to customize pizzas,
place orders, and manage checkout in real time.

## Problem
Small restaurants often need a simple, digital way for customers to place
custom food orders without relying on third-party delivery apps.

## Solution
This project provides a web-based pizza ordering system with customizable
menus, cart functionality, and order management.

## Features
- Browse pizza menu
- Customize pizza size & toppings
- Add/remove items from cart
- Real-time price calculation
- Place orders
- Order status tracking

- Admin dashboard for managing orders

## Tech Stack
- Frontend: HTML, CSS, JavaScript 
- Backend:  PHP /laravel
- Database: MySQL 
- Authentication: User authentication and authorization using middleware
                  Secure access to protected routes
                  Role-based access (Admin / User)
